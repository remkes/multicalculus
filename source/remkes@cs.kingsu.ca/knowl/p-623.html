If an integrable function on \(I = [a,b] \times [c,d]\) has the form \(f(x,y) = g(x) h(y)\) for some single variable functions \(g\) and \(h\text{,}\) then it is called separable. (This use of the term ‘separable’ matches with the use of the term for separable differential equations; a differential equation
\begin{equation*} \frac{dy}{dx} = f(x,y) \end{equation*}
is a separable equation if and onl if \(f(x,y)\) is a separable function in the sens od this definition.)
