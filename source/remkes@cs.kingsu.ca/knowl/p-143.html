I want to use the arclength function to reparametrize curve. The process, therefore, has three steps.
  1. I calculate the arclength fuction \(s(t)\) by integration.
  2. I invert the arclength function. Arclength is always an increasing function, so it is always invertible. I write the inverse as \(t(s)\text{.}\)
  3. I use the inverse of the arclength function to reparametrize by replacing \(t\) with \(t(s)\text{.}\)
