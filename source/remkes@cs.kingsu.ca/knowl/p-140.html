Let \(\gamma(t): [a,b] \rightarrow \RR^3\) be a parametric curve with components \((\gamma_1(t), \gamma_2(t), \gamma_3(t))\text{.}\) The arclength function \(s(t): [a,b] \rightarrow [0, \infty)\) is defined by this integral.
\begin{equation*} s(t) = \int_a^t \sqrt{ \left( \frac{d\gamma_1(u)}{du} \right)^2 + \left( \frac{d\gamma_2(u)}{du} \right)^2 + \left( \frac{d\gamma_3(u)}{du} \right)^2 } du \end{equation*}
