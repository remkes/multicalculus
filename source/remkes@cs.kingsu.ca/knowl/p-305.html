Lastly, \(f_3\text{,}\) has an strange exponential. This leads to very strange domain behaviour. If \(y\) is an integer, \(x\) can be any non-zero real number. If \(y\) is a fraction, \(x\) must be positive if the denominator of \(y\) is even, to avoid square roots of negative numbers. If \(y\) is irrational, \(x\) must be positive. This all leads to a very complicated domain in \(\RR^2\text{,}\) something hard to describe directly as a shape.
