In Definition 6.3.4, I defined an integrable function, on an interval, to be a function such that the limit defining the Riemann integral exists. Most usual functions are integrable, particular functions defined on closed intervals so that they don’t diverge near the edges of the interval. There is a whole theory of integrable functions. However, for the purposes of this course, I’m going to work with a slightly restricted class of functions. This won’t affect any applications, since all the reasonable functions I want to use will still be in this category.
