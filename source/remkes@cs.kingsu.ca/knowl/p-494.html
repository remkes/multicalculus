I want to classify critical points: points where \(\nabla f = 0\text{.}\) We can do this informally by looking at each variable individually. If the point is a maximum in all of \(x_1, x_2, \ldots, x_n\text{,}\) then it is a maximum according to our definition above. Likewise, if it is a minimum in all the variables, it is a minimum. If it is a maximum in some of the variables and a minimum in others, it is a saddle point. However, if even one of the variable has no max/min behvaiour (like \(f(x) = x^3\) at \(x=0\)), then the point is neither a maximum, minimum nor saddle point.
