In local coordinates at \((a,b,f(a,b))\text{,}\) this is just matrix multiplication by \(M\text{.}\) So a linear approximation is a matrix multiplication in local coordintaes. What is the matrix \(M\text{?}\) Well, if I work with partial derivatives, the linear approximation should be formed of linear approximation in \(x\) and linear approximation in \(y\text{.}\)
\begin{equation*} f(x,y) \approx f(a,b) + \frac{\del f}{\del x} (a,b) (x-a) + \frac{\del f}{\del x} (a,b) (y-b) \end{equation*}
