Definition 2.2.1.

Let \(\gamma(t): [a,b] \rightarrow \RR^3\) be a parametric curve. I can write \(\gamma(t)\) in it components.
\begin{equation*} \gamma(t) = (x(t), y(t), z(t)) \end{equation*}
The derivative of the parametric curve, \(\gamma^\prime(t)\text{,}\) is calculated by taking the ordinary single-variable derivative of each component.
\begin{equation*} \gamma^\prime(t) = (x^\prime(t), y^\prime(t), z^\prime(t)) \end{equation*}
This derivative, \(\gamma^\prime(t)\text{,}\) produced a vector for each value of \(t\text{.}\) This vector is called the tangent vector to the parametric curve.
