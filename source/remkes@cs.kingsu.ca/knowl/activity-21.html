Activity 2.3.14.

This is a multi-part question on a historically important arclength calculation. (Some historical context is given and the end of the solution).
  1. Find a parametric curve which descrives an ellipse with semimajor axis \(a\) and semiminor axis \(b\text{.}\) (It is easiest for future stpes to orient the larger ellipse axis along the \(x\) axis.)
  2. Set up the arclength calculation to find the circumference of the ellipse.
  3. Factor \(a\) out of the square root and subtract a cosine term so that you can simplify using \((\sin^2 t + \cos^2 t = 1)\text{.}\)
  4. The eccentricity of the ellipse is defined to be the number \(e = \sqrt{1 - \frac{b^2}{a^2}}\text{.}\) Manipulate the integral so that \(e\) shows up under the square root, but \(a\) amd \(b\) are no longer under the square root.
  5. Calculate the integral in the special case \(e=0\text{.}\) (Since this happens only when \(a=b\text{,}\) this is a circle with radius \(a\text{;}\) you should recover the familiar circumference of a circle).
  6. Try to solve the integral for general eccentricity \(0 \lt e \lt 1\text{.}\) Comment on the problems and barriers to integration.
