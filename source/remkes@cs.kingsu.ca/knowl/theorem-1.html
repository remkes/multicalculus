Theorem 6.2.1.

Let \(f: [a,b] \rightarrow \RR\) be a continuous function. Then if \(F(x)\) is any antiderivative of \(f\) (i.e. \(F^\prime(x) = f(x)\)). Any of the following expressions are versions of the Fundamental Theorem of Calculus.
\begin{gather*} \int_a^b f(x) dx = F(b) - F(a)\\ \int_a^b F^\prime(x) dx = F(b) - F(a)\\ \int f(x) dx = F(x) + c \\ \int F^\prime(x) dx = F(x) + c \end{gather*}
All these equations essentially encode the same concept: that integration and differentiation are inverse processes.
