Example 9.1.2.

Consider the constant vector field \(F(x,y) = (0,1)\) on \(\RR^3\text{,}\) as seen in FigureÂ 9.1.3. The vector \((0,1)\) is a unit vector in the vertical y-axis direction in \(\RR^2\text{.}\) This vector fields associates that unit vertical vector to each point in \(\RR^2\) (as a local direction). At each point, the vector points vertically upward from that point. If this field represented a fluid flow, the fluid would be flowing upwards (in \(y\)) at a uniform pace everywhere, following these local direction vectors.
